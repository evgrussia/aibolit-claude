## Что это такое

Aibolit — это MCP-сервер (Model Context Protocol), который превращает Claude в виртуальную медицинскую клинику. После подключения Claude получает доступ к **31 медицинскому инструменту** и может:

- **Принимать пациентов** через ресепшн и направлять к нужному специалисту
- **Консультировать** от имени 35 AI-врачей по всем специализациям медицины
- **Расшифровывать анализы** (ОАК, биохимия, гормоны, онкомаркёры — 67 тестов) с автоматическим выявлением паттернов заболеваний
- **Мониторить витальные показатели** (АД, пульс, SpO2, температура, глюкоза)
- **Искать информацию о лекарствах** в базе FDA и проверять взаимодействия
- **Исследовать литературу** через PubMed и ClinicalTrials.gov
- **Генерировать медицинские документы** (карта, направление, рецепт, выписка)
- **Вести электронные карты** пациентов с историей болезни

## Установка

### Требования

- Python 3.11+
- Claude Desktop или Claude Code

### Шаг 1: Клонировать репозиторий

```bash
git clone <repository-url> aibolit-claude
cd aibolit-claude
```

### Шаг 2: Установить зависимости

```bash
pip install -e .
```

Или вручную:

```bash
pip install mcp httpx pydantic
```

### Шаг 3: Проверить, что всё работает

```bash
python -c "from src.mcp_server import app; print('Aibolit OK')"
```

---

### Для Claude Desktop

Откройте файл настроек Claude Desktop:

- **macOS**: `~/Library/Application Support/Claude/claude_desktop_config.json`
- **Windows**: `%APPDATA%\Claude\claude_desktop_config.json`
- **Linux**: `~/.config/Claude/claude_desktop_config.json`

#### Способ 1: Через `run.py` (рекомендуется, самый надёжный)

```json
{
  "mcpServers": {
    "aibolit-clinic": {
      "command": "python",
      "args": ["C:\\Users\\evgde\\Cursor project\\aibolit-claude\\run.py"]
    }
  }
}
```

`run.py` автоматически настраивает пути — работает на любой ОС без дополнительных настроек.

#### Способ 2: Через модуль с указанием cwd

```json
{
  "mcpServers": {
    "aibolit-clinic": {
      "command": "python",
      "args": ["-m", "src.mcp_server"],
      "cwd": "C:\\Users\\evgde\\Cursor project\\aibolit-claude",
      "env": {
        "PYTHONPATH": "C:\\Users\\evgde\\Cursor project\\aibolit-claude"
      }
    }
  }
}
```

> **Важно**: замените пути на реальный путь к папке проекта на вашем компьютере.
> На macOS/Linux используйте обычные слэши: `/home/user/aibolit-claude`

#### После настройки

Перезапустите Claude Desktop. В интерфейсе чата появится иконка MCP-инструментов (молоток 🔨). Нажмите на неё — вы увидите 31 инструмент клиники Aibolit.

#### Возможные ошибки

| Ошибка | Причина | Решение |
|---|---|---|
| `No module named 'src'` | Неправильный `cwd` или нет `PYTHONPATH` | Используйте **Способ 1** (`run.py`) |
| `No module named 'mcp'` | Не установлены зависимости | `pip install mcp httpx pydantic` |
| `Server disconnected` | Python не найден | Укажите полный путь: `"command": "C:\\Program Files\\Python311\\python.exe"` |

## Быстрый старт

После подключения MCP-сервера просто общайтесь с Claude на естественном языке. Вот типичные сценарии:

### Сценарий 1: Первое обращение

```
Вы: У меня болит голова уже 3 дня и температура 37.8
```

Claude автоматически вызовет `clinic_reception`, определит, что вам нужен **невролог** и/или **инфекционист**, и начнёт консультацию.

### Сценарий 2: Расшифровка анализов

```
Вы: Расшифруй мои анализы:
    Гемоглобин 105 г/л
    Лейкоциты 12.5
    СОЭ 35
    СРБ 48
    Глюкоза 7.8
    АЛТ 65
    Креатинин 140
```

Claude вызовет `analyze_lab_results` и выдаст:
- Какие показатели в норме, какие нет
- Процент отклонения
- Выявленные паттерны (анемия, воспаление, нарушение функции почек и т.д.)
- Рекомендации по дообследованию

### Сценарий 3: Проверка лекарств

```
Вы: Я принимаю варфарин и хочу выпить ибупрофен от головной боли. Можно?
```

Claude вызовет `check_drug_interactions` и предупредит о критическом взаимодействии.

### Сценарий 4: Регистрация и ведение пациента

```
Вы: Зарегистрируй пациента: Иванов Пётр, 15.03.1980, мужчина, группа крови A+
    Аллергия на пенициллин.
    Запиши давление 150/95, пульс 88, температура 36.6
```

## Все инструменты (31 MCP Tool)

### 1. Навигация по клинике

#### `clinic_reception` — Ресепшн

Первая точка контакта. Анализирует жалобы и рекомендует специалиста.

```
Пример запроса в чате:
"У меня болит в груди при нагрузке и одышка"

→ Система рекомендует: Кардиолог
```

| Параметр | Тип | Описание |
|---|---|---|
| `query` | string, **обязательный** | Жалобы или запрос пациента |

---

#### `list_doctors` — Список врачей

Показывает всех врачей, сгруппированных по категориям (терапевтические, хирургические, диагностические, специализированные, экстренные).

| Параметр | Тип | Описание |
|---|---|---|
| `specialty_filter` | string, опционально | Фильтр по названию специальности |

```
"Покажи всех хирургов"
"Какие врачи есть в клинике?"
```

---

#### `consult_doctor` — Консультация врача

Запускает консультацию у конкретного AI-врача. Врач получает контекст: свои навыки, рекомендуемые анализы, коды МКБ по профилю, данные пациента (если есть).

| Параметр | Тип | Описание |
|---|---|---|
| `specialty` | string, **обязательный** | ID специальности (см. таблицу выше) |
| `complaints` | string, **обязательный** | Жалобы пациента |
| `patient_id` | string, опционально | ID пациента в системе |

```
"Запиши меня к кардиологу, беспокоит давление 160/100 и головокружение"
"Хочу проконсультироваться с эндокринологом по поводу щитовидной железы"
```

### 2. Диагностика

#### `analyze_lab_results` — Расшифровка анализов

Самый мощный диагностический инструмент. Анализирует панель из любого количества тестов, сравнивает с нормами и **автоматически выявляет клинические паттерны**:

- Железодефицитная / макроцитарная / нормоцитарная анемия
- Цитолитический синдром (поражение печени)
- Нарушение функции почек
- Воспалительный синдром
- ДВС-синдром
- Сахарный диабет / преддиабет
- Гипо-/гипертиреоз

| Параметр | Тип | Описание |
|---|---|---|
| `results` | array, **обязательный** | Массив `[{"test": "код_теста", "value": число}]` |
| `gender` | string | `"male"` или `"female"` (по умолчанию `"male"`) |

**Доступные коды тестов (67 штук):**

| Категория | Коды тестов |
|---|---|
| **ОАК** | `hemoglobin`, `erythrocytes`, `leukocytes`, `platelets`, `esr`, `hematocrit`, `mcv`, `mch`, `mchc`, `reticulocytes` |
| **Лейкоформула** | `neutrophils`, `lymphocytes`, `monocytes`, `eosinophils`, `basophils` |
| **Биохимия** | `glucose_fasting`, `glucose_postprandial`, `hba1c`, `total_cholesterol`, `ldl`, `hdl`, `triglycerides`, `total_bilirubin`, `direct_bilirubin`, `alt`, `ast`, `alkaline_phosphatase`, `ggt`, `total_protein`, `albumin`, `creatinine`, `urea`, `uric_acid`, `crp` |
| **Электролиты** | `sodium`, `potassium`, `chloride`, `calcium`, `magnesium`, `phosphorus` |
| **Железо** | `iron`, `ferritin` |
| **Коагулограмма** | `pt_time`, `inr`, `aptt`, `fibrinogen`, `d_dimer` |
| **Гормоны** | `tsh`, `free_t4`, `free_t3` |
| **Онкомаркёры** | `psa`, `cea`, `ca125`, `ca199`, `afp` |
| **Почки** | `gfr` |
| **Моча** | `urine_ph`, `urine_specific_gravity` |

> Для тестов с гендерными нормами (гемоглобин, креатинин, железо, ферритин и др.) система автоматически подбирает нужный диапазон по параметру `gender`.

```
Пример:
"Расшифруй анализы, пол женский:
 Гемоглобин 98, MCV 72, железо 5.2, ферритин 8,
 лейкоциты 11.2, СРБ 45, глюкоза натощак 6.8, ТТГ 6.5"

Результат:
  ↓ Гемоглобин: 98 г/л (норма 120–150) — АНЕМИЯ
  ↓ MCV: 72 фл (норма 80–100) — микроцитоз
  ↓ Железо: 5.2 мкмоль/л (норма 9.0–30.4)
  ↓ Ферритин: 8 мкг/л (норма 10–120)
  ↑ Лейкоциты: 11.2 ×10⁹/л (норма 4.0–9.0)
  ↑ СРБ: 45 мг/л (норма 0–5)
  ↑ Глюкоза: 6.8 ммоль/л (норма 3.9–5.5)
  ↑ ТТГ: 6.5 мМЕ/л (норма 0.4–4.0)

  Паттерны:
  • Железодефицитная анемия (↓Hb, ↓MCV, ↓Fe)
  • Воспалительный синдром (↑СРБ, ↑лейкоциты)
  • Нарушение гликемии натощак (5.6–6.9) — преддиабет
  • Субклинический гипотиреоз (↑ТТГ, нормальный свТ4)
```

#### `assess_vitals` — Оценка витальных показателей

| Параметр | Тип | Описание |
|---|---|---|
| `systolic_bp` | integer | Систолическое АД (мм рт.ст.) |
| `diastolic_bp` | integer | Диастолическое АД (мм рт.ст.) |
| `heart_rate` | integer | ЧСС (уд/мин) |
| `temperature` | number | Температура (°C) |
| `spo2` | number | Сатурация кислорода (%) |
| `respiratory_rate` | integer | Частота дыхания (/мин) |
| `blood_glucose` | number | Глюкоза крови (ммоль/л) |

## FAQ

### Нужен ли API-ключ?
Нет. Все внешние API бесплатные и не требуют регистрации. PubMed просит не превышать 3 запроса/сек (система это соблюдает).

### Система сохраняет данные пациентов?
Да, в файлах `data/patients/*.json` на вашем компьютере. Данные НЕ отправляются на внешние серверы (кроме запросов к Claude и медицинским API по диагностике).

### Можно ли использовать для реальной диагностики?
**Нет!** Это информационная система для образовательных целей. Все рекомендации AI требуют верификации реальным врачом.

### Как добавить нового врача-специалиста?
Добавьте новую `Specialization` в `src/agents/specializations.py` по образцу существующих.

### На каком языке писать запросы?
На любом. Система работает на русском и английском. Для поиска в PubMed лучше использовать английские термины.

### Как обновить нормы анализов?
Отредактируйте `LAB_REFERENCE_RANGES` в `src/models/medical_refs.py`.
