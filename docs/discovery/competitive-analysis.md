# Aibolit AI — Конкурентный анализ

## 1. Обзор рынка медицинских AI-сервисов

### Глобальный рынок

Рынок AI в здравоохранении демонстрирует устойчивый рост. Основные драйверы:

- **Дефицит врачей:** ВОЗ прогнозирует нехватку 10 млн медицинских работников к 2030 году
- **Рост стоимости здравоохранения:** пациенты ищут более доступные способы первичной оценки симптомов
- **Цифровизация:** COVID-19 ускорил принятие телемедицины и цифровых инструментов здоровья
- **Развитие LLM:** GPT-4, Claude, Gemini значительно повысили качество медицинских AI-ответов

Ключевые сегменты:
1. **Symptom checkers** — оценка симптомов (Ada Health, Symptomate)
2. **AI-консультации** — диалоговые AI-врачи (Babylon Health, K Health)
3. **Телемедицина с AI** — AI как ассистент врача (DocDoc, Сбер Здоровье)
4. **Специализированные AI** — дерматология, радиология, психиатрия

### Российский рынок

Особенности российского рынка медицинских AI-сервисов:

- **Объём:** рынок цифрового здравоохранения в России оценивается в 100+ млрд руб. (2024), из них AI-сегмент — около 5-10 млрд руб.
- **Регулирование:** 152-ФЗ (персональные данные), ФЗ-323 (врачебная тайна), отсутствие специального регулирования medical AI
- **Языковой барьер:** большинство мировых AI-решений работают на английском; русскоязычных аналогов с доказательной базой мало
- **Доверие:** низкое доверие к AI-диагностике (по опросам — 30-40% готовы попробовать)
- **Государственная инициатива:** ЕГИСЗ, ЕМИАС — но без AI-компонента для пациентов
- **Конкуренция:** преимущественно телемедицинские платформы (СберЗдоровье, DocDoc, Яндекс Здоровье) — AI как вспомогательный инструмент, не основной

## 2. Анализ конкурентов

### 2.1. Ada Health (Германия)

| Параметр | Значение |
|----------|----------|
| Страна | Германия (Берлин) |
| Год основания | 2011 |
| Модель | Freemium (базовый — бесплатно, Premium — $5/мес) |
| Языки | Английский, немецкий, испанский, португальский, румынский, суахили |
| Русский язык | Нет |
| Технология | Proprietary Bayesian network + NLP |
| Специализации | Не выделены (общий symptom checker) |
| Пользователи | 13 млн+ (заявлено) |

**Сильные стороны:**
- Клинически валидирована (публикации в BMJ, Nature Medicine)
- Обширная база из 10 000+ заболеваний
- Партнёрства с NHS (Великобритания), Bayer
- Прошла CE-маркировку как медицинское изделие класса I

**Слабые стороны:**
- Нет русского языка
- Не диалоговый формат — пошаговый опросник (не чат)
- Нет мультитернового контекста
- Не указывает уровни доказательности

- Комбинация AI + живые врачи (телемедицина)

- Модуль мониторинга здоровья (wearables)

**Слабые стороны:**
- Финансовые проблемы (реструктуризация в 2023-2024)
- Дорогая подписка для B2C
- Нет русского языка
- AI — только для триажа, основная ценность — живые врачи
- Критика за точность AI-диагностики (BMJ Open, 2020)

- Доступен только в США
- Ограниченное число специализаций (5 vs 35 у Aibolit)

## 4. Конкурентные преимущества Aibolit AI

### 4.1. Уникальные преимущества

1. **35 медицинских специализаций с AI-врачом.** Ни один конкурент не предлагает AI-консультации по 35 направлениям. Ближайший — K Health с 5 специализациями. Это позволяет пациенту получить узкоспециализированную информацию (кардиолог, нефролог, генетик, гериатр и т.д.) без ожидания очереди к живому врачу.

2. **Полностью бесплатный.** Единственный продукт, сочетающий глубину AI-консультаций (мультитерновый чат + диагностические инструменты + лекарства) с полной бесплатностью. Ada и Symptomate бесплатны, но ограничены простым опросником. K Health бесплатен для AI-триажа, но платный для врачей.

3. **Нативный русский язык с медицинской терминологией.** В отличие от зарубежных аналогов (Ada, Babylon, K Health), Aibolit AI создан для русскоязычных пациентов. Дисклеймеры, red flags, формулировки — всё на русском, с учётом российской медицинской практики.

4. **Диагностический комплекс.** Расшифровка лабораторных анализов, оценка витальных, расчёт СКФ, кардиоваскулярный риск — ни один конкурент не предлагает такой набор инструментов бесплатно.

5. **Интеграция с доказательной базой.** PubMed, OpenFDA, RxNorm, WHO ICD-11, ClinicalTrials.gov — AI-ответы подкрепляются реальными научными источниками. Это уникально для бесплатного продукта.

6. **8 типов медицинских дисклеймеров.** Самая проработанная система безопасности среди конкурентов: дисклеймеры для диагностики, лечения, лекарств, анализов, визуализации, экстренных случаев, педиатрии.

## 6. Выводы и рекомендации

### Позиционирование

Aibolit AI занимает **уникальную нишу**: бесплатный AI-медицинский ассистент с максимальной глубиной (35 специализаций + диагностические инструменты) для русскоязычного рынка. Прямых конкурентов с таким набором характеристик **не существует**.

### Стратегические рекомендации

1. **Фокус на безопасность (Q1).** Закрыть пробелы (шифрование, аудит, тесты) до публичного запуска. Это критично для доверия и соответствия 152-ФЗ.

2. **PWA и мобильность (Q2).** 70%+ медицинских поисковых запросов — с мобильных устройств. PWA — необходимое условие для привлечения аудитории.

3. **Клиническая валидация (Q3-Q4).** Провести валидационное исследование для Red Flag Detector и AI-рекомендаций. Публикация результатов повысит доверие.

4. **Партнёрства с НКО (Q2-Q3).** Бесплатность — мощный аргумент для социальных партнёрств (фонды помощи онкобольным, диабетические ассоциации, организации пожилых).

5. **Защита от входа крупных игроков.** Углублять экспертизу по специализациям, расширять диагностический комплекс, строить community. Сетевой эффект через контент (клинрекомендации Минздрав в RAG).
